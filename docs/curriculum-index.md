- 不建议提前连续阅读后面的课程；
- 允许查看全貌，但当前周未通过时不提前堆叠实现；
- 每天教材是完整学习基线，老师会结合你的经验调整免修项；

- Demo 只在进入对应主题时实现，避免生成大量未经验证的样板代码。
