# AI Engineer / AI Product Engineer 求职训练工程

这是你的个人 AI 转型工程。它不是普通入门课，而是面向 6～7 年前端工程师的 AI 工程化与求职作品集训练。

## 当前主线

现在默认使用高级轨道：

[advanced-track/README.md](./advanced-track/README.md)

从这里开始：

[START-HERE.md](./START-HERE.md)

## 新目标

你最终需要拿得出这些证据：

- 一个真实模型 Gateway / AI 调用工程底座；
- 一个结构化输出、校验、重试和评测闭环；
- 一个前端接口契约 RAG 助手；
- 一个 Tool / Agent / MCP 综合 Demo；
- 一个偏产品方向的 AI 睡眠或情绪教练 MVP；
- 一份能用于求职的作品集、简历项目描述和面试问答。

## 节奏

每天最多 2～3 小时。

课程不追求满 48 周，新轨道是 24 周，可以根据实际进度压缩或拉长。

## 协作规则

以后课程默认按你的经验水平处理：

- 跳过低价值 AI 入门解释；
- 只保留对工程、项目、面试有用的知识；
- 每周必须有代码、评测、日志、架构图或作品集证据；
- 所有结论优先沉淀通用规律，不把某个 Demo 案例当成知识本体。
