# 项目三：AI 睡眠教练

周期：第 40～46 周  
项目定位：综合产品能力练习，不作为医疗产品

## 核心场景

帮助希望改善作息的普通用户记录睡眠习惯、发现趋势并制定容易执行的小计划。

## MVP 范围

- [ ] 睡眠日志
- [ ] 睡前目标
- [ ] 每周总结
- [ ] 一次只建议一个小行动
- [ ] 用户反馈建议是否可执行
- [ ] 删除个人数据

## 安全边界

- 不诊断失眠或其他疾病；
- 不提供药物或剂量建议；
- 不承诺治疗效果；
- 遇到持续严重症状时建议咨询专业人士；
- 清晰说明数据用途和删除方式。

## 产品评测

- 建议是否具体；
- 建议是否基于用户记录；
- 是否避免过度承诺；
- 用户是否实际执行；
- 一周后是否愿意继续使用。

## 毕业标准

- [ ] 完成一轮七天体验
- [ ] 至少五名测试用户
- [ ] 有隐私和安全说明
- [ ] 有建议质量测试集
- [ ] 有留存或持续使用意愿记录
