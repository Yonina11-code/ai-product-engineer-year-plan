# W15：Agent 可靠性：幂等、重试、人工接管

阶段：工具调用与 Agent

## 为什么学这个

真实 Agent 最大问题是副作用和不可控行为。

## 本周要构建什么

给 Agent 加入幂等键、最大步数、失败重试、人工接管和操作审计。

## 本周核心能力

- 幂等
- 重试策略
- 人工接管
- 审计日志

## 一周节奏（每天 2～3 小时）

- [Day 01：架构判断](../lessons/week-15/day-01.md) — 读代码、画边界、明确这项能力解决什么工程问题。
- [Day 02：最小实现](../lessons/week-15/day-02.md) — 写一个能跑的薄切片，不追求大而全。
- [Day 03：可靠性增强](../lessons/week-15/day-03.md) — 补错误处理、结构校验、边界条件和可观测信息。
- [Day 04：评测与失败分析](../lessons/week-15/day-04.md) — 构造样本集，记录失败原因，避免只看成功演示。
- [Day 05：作品集与面试表达](../lessons/week-15/day-05.md) — 沉淀架构图、代码证据、指标和三分钟讲法。

## 本周交付证据

- [ ] 重复提交防护
- [ ] 失败恢复案例
- [ ] 审计日志样例

## 面试讲法

讲清你如何限制 Agent 的危险行为。

## 通过标准

- 能跑出最小闭环；
- 有失败样本和处理策略；
- 有日志、截图、测试或评测证据；
- 能用自己的项目解释这项能力；
- 不把概念讲成空话。
