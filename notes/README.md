# 学习问答笔记

这里记录学习过程中由学习者主动提出的问题，以及经过老师整理后的回答。

## 和其他目录的区别

- `lessons/`：老师准备的正式教材；
- `reviews/`：学习者完成的作业、实验记录和复盘；
- `notes/`：学习过程中产生的真实问题、阶段结论和延伸知识。

## 记录规则

- 按周建立问答文件，例如 `week-01-qa.md`；
- 保留问题原意，不把问题改造成抽象术语；
- 回答先给直接结论，再解释真实工程中的做法；
- 标注这道问题对应的知识点；
- 对尚未实践验证的内容标记为“待实验”；
- 同一个问题有新的理解时，在原记录下继续补充，不重复创建多份答案。
